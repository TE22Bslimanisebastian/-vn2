Lorem Ipsum

Vi kan skapa utfullnadstext i VS Code med att skriva Lorem och sedan trycka RETUR

Detta ger 20 utfyllnadsord

Vi kan bestämma hur många

Vi kan bestämma hur många utfyllnadsord genom att skriva LoremNR exempelvis lorem6 ger 6 ord

Om vi vill ha väldigt många ord så kan vi skriva lorem *NR vilket multiplicera NR med 20

Exempelvis om vi skriver lorem *3 får vi 20*3 ord
